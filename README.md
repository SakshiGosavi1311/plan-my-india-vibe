# 🇮🇳 AI Trip Planner for India

An AI-powered travel planning web application that helps users create personalized travel itineraries across India based on their destination, budget, travel duration, and interests.

🔗 **Live Demo:** https://plan-my-india-vibe.vercel.app/

---

## ✨ Features

- 🤖 AI-generated travel itineraries
- 📍 Destination recommendations
- 💰 Budget-based planning
- 🗓️ Trip duration customization
- 🎯 Interest-based suggestions
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/plan-my-india-vibe.git
```

### Navigate to project

```bash
cd plan-my-india-vibe
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The application will run locally at:

```
http://localhost:5173
```

## 💡 How It Works

1. User enters destination.
2. Selects travel duration.
3. Sets budget.
4. Chooses travel interests.
5. AI generates a personalized travel itinerary.
